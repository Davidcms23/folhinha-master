Sobre o projeto

ProbPath é uma trilha de 30 dias de flashcards para memorizar as principais distribuições de probabilidade usando o Anki.

A progressão foi planejada por dependência conceitual: você começa com os fundamentos (variância, FGM, assimetria) e vai avançando da distribuição de Bernoulli até a distribuição Beta, sempre construindo sobre o que já aprendeu.

Cada dia tem exatamente 10 cards. O botão Copy Anki text copia o par pergunta/resposta no formato correto para importar no Anki via File → Import (separador: tab).

Distribuições cobertas

  • Fundamentos (variância, FGM, assimetria, curtose, séries)
  • Bernoulli e Binomial
  • Uniforme contínua
  • Exponencial
  • Normal
  • Gama e Qui-Quadrado
  • Beta e distribuição F
  • Revisão de relações entre distribuições

Estrutura do projeto

  • index.html — página principal
  • cards.js — todos os cards (fácil de editar e expandir)
  • app.js — lógica de interação e localStorage
  • style.css — estilos com suporte a dark mode
