Como configurar o Anki

1. Instalar o addon MathJax

Para renderizar LaTeX nos cards, instale o addon MathJax for Anki (ID: 1280253613) via Ferramentas → Complementos → Obter complementos.

2. Copiar os cards

Para cada card que quiser adicionar:

  1. Abra o dia desejado na trilha
  2. Clique em Copy Anki text (LaTeX included)
  3. Cole em um arquivo .txt (um card por linha)

3. Importar no Anki

  1. No Anki: Arquivo → Importar
  2. Selecione o arquivo .txt
  3. Separador: Tab
  4. Campo 1 → Frente, Campo 2 → Verso
  5. Clique em Importar

4. Formato do LaTeX no Anki

Os cards usam a sintaxe padrão do MathJax:

  • \( ... \) para equações inline
  • \[ ... \] para equações em bloco

Se preferir o addon LaTeX nativo do Anki, substitua por [latex] ... [/latex].

Dica

O botão LaTeX only copia só a fórmula — útil para colar diretamente num card que você já criou manualmente.
